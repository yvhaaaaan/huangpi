# 1. 项目说明

黄陂镇油茶客家文旅非遗小程序是一款文旅导览类微信小程序。它不是电商平台，主要目标是让游客快速了解黄陂镇有什么特色、在哪里、怎么玩、有哪些活动可以参加。

前端页面需要做到结构清楚、入口明确、图片区域有占位、内容为空时不乱版。普通用户主要浏览和报名，商家可以在小程序内上传油茶与特产内容，政府/系统管理员负责审核和平台管理。

# 5. 全局前端规则

## 5.1 登录规则

游客可以浏览公开内容，包括首页、文化内容、地图、路线和活动详情。

点击收藏、活动报名、我的收藏、我的报名时，如果未登录，需要弹出登录提示。

登录成功后，应返回用户刚才操作的页面，尽量不要让用户重新寻找入口。

个人中心根据账号角色显示不同管理入口，普通用户不显示商家管理和平台管理。

## 5.2 图片占位规则

首页轮播图、卡片封面、详情页轮播图、活动封面、特产图片都需要预留固定图片区域。

后台未上传图片时，前端显示默认占位图。

图片未审核通过时，前台不展示真实图片。

图片加载失败时，显示默认占位图，避免页面空白或布局错乱。

## 5.3 内容状态规则

| 状态 | 含义 | 前端展示规则 |
| --- | --- | --- |
| 草稿 | 内容还未提交 | 只在管理页面显示，普通前台不显示。 |
| 待审核 | 商家提交后等待审核 | 只在管理页面显示，普通前台不显示。 |
| 已发布 | 审核通过 | 前台可以展示。 |
| 已驳回 | 审核未通过 | 商家管理页面显示驳回原因，前台不显示。 |
| 已下架 | 内容不再展示 | 前台不显示，收藏列表可提示“内容已下架”。 |

## 5.4 收藏规则

登录用户可以收藏油茶文化、油茶与特产、景点、非遗项目、路线和活动。

未收藏时显示普通收藏图标，已收藏时图标高亮。

未登录点击收藏时，弹出登录提示。

内容下架后，收藏列表中可隐藏该内容，或显示“内容已下架”。

## 5.5 查看更多规则

首页各推荐栏目默认展示最多 3 条。

油茶与特产展示页首页区域默认展示前 3 条。

当内容超过 3 条时，显示“查看更多”按钮。

点击“查看更多”进入对应列表页，展示全部已发布内容。
